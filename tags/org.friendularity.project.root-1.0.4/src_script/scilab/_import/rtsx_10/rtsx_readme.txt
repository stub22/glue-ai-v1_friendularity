RTSX (Robotic Tools for Scilab/Xcos) Ver 1.00 Beta
==================================================
www.controlsystemslab.com  August, 2012

This project aims at building a Scilab/Xcos tools for robotics field. It is
currently under development. To start using the toolset, enter

exec('startup_rtsx.sce',-1)

at Scilab command prompt. 

To check whether RTSX works properly, type

rprdemo

in Scilab console to see plots and animation of an RPR robot.


Online help is available at

http://controlsystemslab.com/rtsx/online-help/
